01

先搞懂3个基本概念

📖
什么是期权？
What is an Option?
⌄

期权是一种「权利合约」。你花一笔钱（叫做权利金/Premium），买到一个「在未来某天，以固定价格买或卖某只股票」的权利。

关键词：
· 行权价（Strike Price）：合约里约定的固定买卖价格
· 到期日（Expiration Date）：这个权利的有效期
· 权利金（Premium）：你付的"购买这个权利"的费用，也是你买方的最大亏损

💡 类比：你花500元买了一张"房子预购权"，约定3个月内可以用100万买某套房。如果3个月内房价涨到120万，你行使这个权利赚了20万；如果房价跌了，你只损失500元权利金，不强制买房。
📈
Call vs Put — 一句话区别
Call vs Put
⌄

Call（认购期权）：赋予你「以行权价买入」股票的权利。
→ 你看涨才买 Call。股票涨，Call 涨价，你赚钱。

Put（认沽期权）：赋予你「以行权价卖出」股票的权利。
→ 你看跌才买 Put。股票跌，Put 涨价，你赚钱。

买 Call 赚钱条件
股价 大幅上涨
买 Put 赚钱条件
股价 大幅下跌
买方最大亏损
只损失权利金
⚖️
买方 vs 卖方 — 谁风险更大？
Buyer vs Seller
⌄

买方（Long）：付权利金，获得权利但没有义务。最大亏损 = 权利金，潜在收益可以很大。

卖方（Short/卖出）：收权利金，但有义务履约。最大收益 = 权利金，潜在亏损可能很大甚至无限（裸卖Call）。

⚠️ 小白重点：先从「买 Call」「买 Put」开始练手。卖方策略（尤其裸卖）亏损可能远超预期，需要充分理解后再操作。
02

8种策略逐一详解

📈
① 买入 Calls（Long Call）
Long Call
看涨 ✓ 新手友好
⌄

一句话理解

我觉得这只股票要涨，花一笔小钱，买到一个「以固定低价买入」的权利，大涨时赚大钱。

盈亏结构

最大收益
理论无限
最大亏损
权利金（全亏）
盈亏平衡点
行权价 + 权利金

具体操作步骤

  1. 确认自己看涨该股票，且有明确的涨幅预期（如未来4周涨10%以上）
  2. 查看当前股价，选一个「价外一档」的行权价（比股价高5%左右）
  3. 选择到期日：短期交易选2~4周后，长期持仓选2~3个月后
  4. 查看 IV（隐含波动率）百分位，最好 < 40%，说明期权便宜
  5. 买入，权利金就是你的最大亏损
  6. 设止损：当权利金亏损 40~50% 时平仓离场

实例

苹果股价$180，你买一张行权价$185、4周后到期的Call，权利金$3。
· 到期时苹果 $200：你盈利 $200-$185-$3 = $12（每股）
· 到期时苹果 $180：Call 归零，亏损 $3 权利金

时间是你的敌人：买入 Call 后，每过一天都会因为时间衰减（Theta）损失一点价值。不要持仓太久不管。
📉
② 买入 Puts（Long Put）
Long Put
看跌 对冲持仓 ✓ 新手友好
⌄

一句话理解

我觉得股票要跌，或者我已经持有正股想保险，买 Put 来做对冲。股票越跌，Put 越值钱。

两种用法

用法A - 方向做空：不持有股票，纯粹做空押注。
用法B - Protective Put（保护型）：已经持有正股 + 同时买入 Put。相当于给股票买了「保险」，股票跌了 Put 赚钱抵消损失。

盈亏结构

最大收益
行权价 - 权利金（股票跌到0）
最大亏损
权利金（全亏）
盈亏平衡点
行权价 - 权利金

注意

⚠️ 市场大跌时 IV 暴涨，Put 会变贵：在大跌时追买 Put（比如 VIX 已经飙升到 30+ 时），往往买在最贵的时候。建议在市场平静、IV 低时预先买入保护性 Put。
💰
③ 正股担保卖 Calls（Covered Call）
Covered Call
收租金 温和看涨/横盘 ✓ 新手友好
⌄

一句话理解

你已经持有100股，在此基础上卖出一张 Call，收取权利金当「租金」。如果股票没涨过行权价，权利金就是净赚；如果大涨超过行权价，股票被强制卖出，你赚了股票涨幅+权利金但错过更大的涨幅。

盈亏结构

最大收益
(行权价-成本价) + 权利金
最大亏损
股票跌到0（和持股一样）
盈亏平衡点
持股成本 - 权利金

操作步骤

  1. 已持有至少100股某只股票
  2. 选一个比当前股价高 5~10% 的行权价（你接受在这个价格卖股票）
  3. 选择 30~45 天到期（时间价值衰减最快）
  4. 卖出1张对应的 Call，收到权利金
  5. 到期前如果没被行权：再次卖出新的 Call，继续收租
  6. 如果股价逼近行权价：可以买回 Call 平仓（buyback），防止被强制卖股
💡 适合场景：你长期持有某只股票，短期内判断不会大涨，希望每月多一点现金流收入。
❌
④ 现金担保卖 Puts（Cash-Secured Put）
Cash-Secured Put — 当前被锁定
收租金 温和看涨
⌄

一句话理解

提前在账户备好购买股票所需的全额现金，然后卖出 Put。相当于：「我愿意在某个更低的价格买入这只股票，等待期间先收权利金」。

· 股票没跌到行权价 → 权利金全赚，不需要买股票
· 股票跌破行权价 → 被迫以行权价买入股票（用之前准备好的现金）

为何当前被锁定？

通常是账户资金量未达到要求。现金担保卖 Put 需要账户里有足够的现金（等于行权价×100×合约数）。例如行权价$50，需要备好$5,000现金。建议联系券商确认解锁条件。

和保证金担保卖Put的区别

现金担保 = 全额备好现金，没有杠杆，风险低
保证金担保 = 只需一部分保证金，有杠杆，风险更高

⚡
⑤ 空仓/保证金担保卖 Puts
Margin-Secured / Naked Put
收租金 温和看涨 ⚠ 需经验
⌄

一句话理解

和现金担保卖 Put 逻辑相同，但不需要备足全额现金，只需要一部分保证金。杠杆更高，收益放大，但如果股票大跌，亏损也会更大。

盈亏结构

最大收益
收取的权利金
最大亏损
行权价 × 100 - 权利金（股票归零）
盈亏平衡点
行权价 - 权利金
⚠️ 小白警告：卖 Put 方向判断错误时亏损可能很大。例如行权价$50卖出Put，股票跌到$30，每股亏$20，100股就是$2000，远超收到的权利金。必须设好止损。

风险控制要点

  1. 只在你真正愿意持有的股票上卖 Put（即使被迫接货也不亏心）
  2. 行权价选在重要支撑位下方（技术面支撑）
  3. 亏损超过权利金 2倍时坚决平仓
  4. 单笔保证金占用 ≤ 账户净值 20%
↕️
⑥ 垂直价差策略（Vertical Spread）
Bull Call Spread / Bear Put Spread
看涨版 看跌版 ✓ 新手友好
⌄

一句话理解

同时买一张期权 + 卖一张相同到期日但不同行权价的期权。两腿对冲后，成本更低、风险有限、收益也有限。适合有方向判断但不想花太多权利金的情况。

看涨版：Bull Call Spread（牛市Call价差）

  1. 买入较低行权价的 Call（比如行权价 $180）
  2. 同时卖出较高行权价的 Call（比如行权价 $190），收回部分权利金
  3. 净成本 = 买入的权利金 - 卖出的权利金
  4. 最大收益 = 行权价之差 - 净成本；最大亏损 = 净成本

看跌版：Bear Put Spread（熊市Put价差）

  1. 买入较高行权价的 Put（比如行权价 $180）
  2. 同时卖出较低行权价的 Put（比如行权价 $170）
  3. 净成本更低，但收益也封顶
💡 为什么用价差而不是直接买Call/Put？
直接买Call成本$5，全错了亏$5；Bull Call Spread净成本$2，错了最多亏$2，同时也降低了对IV高低的依赖。适合IV偏高时。

实例（Bull Call Spread）

苹果股价 $180：
买 $185 Call，花 $4；卖 $195 Call，收 $1.5。净成本 $2.5。
苹果到期涨到 $200：最大收益 $(195-185) - $2.5 = $7.5
苹果到期跌到 $178：最大亏损 $2.5（净权利金）

🗓️
⑦ 日历 / 对角线价差策略
Calendar Spread / Diagonal Spread
横盘/低波动 赚时间价值 ⚠ 需理解Theta
⌄

日历价差：一句话理解

卖出近月期权（快速衰减）+ 买入远月相同行权价期权（衰减慢）。赚的是两者时间价值衰减速度的差异。

为什么近月衰减比远月快？

期权的时间价值在到期前的最后几周衰减最快（就像冰淇淋在气温最高时化得最快）。近月 Theta 大，卖出后收益更高；远月 Theta 小，持有成本低。

对角线价差

在日历价差基础上，买卖的行权价也不同。更灵活，但理解成本更高，建议先掌握日历价差再进阶。

⚠️ 关键风险：适合横盘低波动市场。如果股票大涨大跌，两腿都可能亏损。
另外：在财报前后 IV 变化剧烈，日历价差会受到 Vega 影响较大，要注意。
🦅
⑧ 铁鹰 / 宽跨式（Iron Condor / Strangle）
Iron Condor / Short Strangle
区间震荡 收权利金 ⚠ 进阶策略
⌄

一句话理解

同时卖出一个价外 Call 和一个价外 Put，赌股票在到期前「不会大涨也不会大跌」，收取两边的权利金。

Iron Condor = 宽跨式 + 在两侧各买一个更价外的期权作为保护，变成风险有限版本。

适合什么时候用？

· 股票长期在一个箱体震荡
· IV 较高（权利金丰厚），但你判断不会有大行情
· 典型使用场景：标普500 ETF（SPY）的月度日历

盈亏结构（Iron Condor）

最大收益
收到的净权利金
最大亏损
价差宽度 - 净权利金
安全区间
两个行权价之间
⚠️ 突破风险：一旦股票向任意方向突破行权价，亏损迅速放大。必须设好止损，价格接近边界时果断平仓一侧。
03

不同行情 → 用什么策略

📈 单边上涨

连续阳线，均线多头排列，ADX > 25

  • 首选：买入 Call（杠杆做多）
  • 温和上涨：Bull Call Spread（降低成本）
  • 持股增强：Covered Call（收租）
  • 回调接货：保证金卖 Put
❌ 不要：买 Put 做空，大概率全亏
📉 单边下跌

均线空头排列，MACD 死叉，VIX 上升

  • 首选：买入 Put（做空）
  • 持股保险：Protective Put（正股+买Put）
  • 降低成本：Bear Put Spread
❌ 不要：裸卖 Put（大跌中接货亏死）
↔️ 区间震荡

价格在支撑/阻力之间来回，ATR 低

  • 首选：Iron Condor（两边收权利金）
  • 简单版：日历价差（赚时间衰减）
  • Covered Call（持股收租）
❌ 不要：直接买 Call 或 Put（时间价值每天损耗）
⚡ 财报/重大消息前

IV 极高，方向不明，短期波动巨大

  • 首选：Long Straddle（买跨式，两头押）
  • 成本高时：Long Strangle（买宽跨式）
  • 提前低IV时布局最优
❌ 不要：任何卖方策略（消息落地前风险极大）

希腊字母白话解释

字母 叫什么 白话解释 实战怎么用
Δ
Delta		 Delta
德尔塔		 股价每涨$1，期权价格变化多少。
买 Call: +0~1；买 Put: -1~0		 Delta=0.5意味着股价涨$1，你的Call涨$0.5。看涨选Delta 0.4~0.6 的Call。
Θ
Theta		 Theta
时间损耗		 每过一天，期权价值减少多少（对买方是成本，对卖方是收益）。 买期权Theta是负的（每天亏一点）。不能持有太久。卖期权Theta是正的（每天赚一点）。
ν
Vega		 Vega
波动率敏感度		 隐含波动率(IV)每涨1%，期权价值变化多少。 财报前IV飙升，买期权Vega正收益；财报后IV暴跌（IV Crush），期权价值大跌——即使方向对了也可能亏。
Γ
Gamma		 Gamma
德尔塔加速度		 Delta 变化的速度。平价期权 Gamma 最大。 近到期的平价期权Gamma巨大，价格可能快速剧烈变化。临近到期时要格外小心。
ρ
Rho		 Rho
利率敏感度		 利率变化对期权价格的影响。 日常交易通常可忽略，只有长期LEAPS期权需要关注。
💡 小白只需记住3个：Delta（方向）、Theta（时间损耗）、Vega（波动率影响）。这三个就能覆盖90%的期权分析需求。
05

每次下单前的7步检查

  • 01
    判断当前市场状态
    上涨？下跌？震荡？高波动？均线方向、ADX 是否 > 25、最近K线形态。
  • 02
    查看 IV 百分位
    IV 百分位 < 30%：期权便宜 → 偏向买入期权。
    IV 百分位 > 70%：期权贵 → 偏向卖出期权（价差）或买跨式。
  • 03
    确认近期是否有财报/重大消息
    财报前后 IV 会剧烈变化。不了解 IV Crush 效应前不要在财报前卖期权。
  • 04
    选择合适的到期日
    方向交易：2~4 周后。时间价值策略（Covered Call 等）：30~45 天。事件驱动：事件后 1~2 天。
  • 05
    计算最大亏损 / 最大收益 / 盈亏平衡点
    在下单前就知道最坏情况是多少钱。买方：最大亏损 = 权利金；卖方：根据保证金计算。
  • 06
    单笔风险控制
    买期权：单笔权利金 ≤ 账户总值 2%。卖期权：保证金占用 ≤ 账户净值 20%。
  • 07
    设置退出条件（提前想好）
    止损：权利金亏损 40~50% 就平仓。止盈：盈利 50~100% 可考虑先兑现部分。时间止损：距到期不足 1 周且无行情，平仓不赌。
🔑 最终原则：任何策略在你不能用一句话清晰解释「我的最大亏损是多少、什么情况下会发生」之前，不要动用真实资金。可以先用纸上交易（Paper Trade）练手。
