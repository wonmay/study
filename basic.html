销售常见疑问 · 标准答案 8 条

客户问到这些时，你直接用这些话回答——零基础也能对答如流

Token 和字数有什么关系？怎么估算费用？

简单记忆法：1个中文字 ≈ 1.5 个Token，1个英文单词 ≈ 1 个Token。

估算公式：月均字符数 ÷ 1.5 = 月均 Token 数，再乘以单价就是费用。大多数中小企业月消耗在 10万～1000万 Token 之间，月费用大概在几百到几万元不等。

💡 销售技巧：直接问客户"你们现在每月在 AI 上花多少钱"，从费用反推 Token 量，比讲概念更有效。

客户问"TPM 是什么"，怎么用大白话解释？

TPM = 每分钟能处理多少个 Token，就像高速公路的车道数：车道越多，车跑得越快、不堵车。

举个例子：TPM = 100万，意味着 AI 每分钟可以同时处理 100万字的请求。官网共享集群的 TPM 不稳定，高峰期可能只有 10万；rhcloud 专有集群保障在高 TPM 档位，吐字速度稳定。

RPD 超限会怎样？有没有办法解决？

RPD（每天请求次数上限）超限后，所有 API 请求会被直接拒绝、返回错误，用户侧表现就是"AI 不回复了"。

官网免费额度的 RPD 通常很低（如几千次/天），付费套餐高些但也有限制。rhcloud 专有集群的 RPD 远高于官网共享资源，且可按业务增长扩容，不会因为 RPD 卡脖子。

客户说"我用的是 ChatGPT，为什么要换？"

个人用 ChatGPT 没问题，企业生产环境用 ChatGPT 有三大风险：

  • ① 合规风险：数据出境，国内监管合规要求无法满足
  • ② 稳定性风险：国内访问需要工具，高峰期不稳定，影响业务
  • ③ 成本风险：GPT-4 价格远高于国内同等能力模型，性价比低

💡 切入话术：「您现在是用个人账号还是企业 API？如果是生产环境，我们可以帮您做一个合规+降本的对比方案。」

开源模型和闭源模型，销售要怎么区分？

一句话区别：开源 = 可以下载自己部署，闭源 = 只能通过 API 调用付费使用。

对销售来说，这个技术细节不重要。关键是：客户选哪个，取决于成本和效果平衡。DeepSeek 是开源但需要自建算力；GPT-4o 是闭源但效果顶尖成本高。rhcloud 把两类都接了，帮客户按需自动路由。

什么是上下文窗口？客户为什么关心这个？

上下文窗口 = AI 一次能记住的对话长度。就像人的短期记忆，超过这个限制，AI 就"忘"了之前说的话。

客户关心它，因为：① 做长文档分析，窗口不够就分析不完整；② 做多轮客服对话，窗口短了 AI 会答非所问；③ 窗口越大，单次消耗 Token 越多，成本越高。选模型时要结合实际场景匹配窗口大小。

什么是RAG？和普通调用 AI 有什么区别？

RAG = 让 AI 先查资料再回答，而不是只靠自己的"记忆"。

比喻：普通 AI 是闭卷考试，靠训练数据；RAG 是开卷考试，考前先检索文档。客户企业有大量内部知识库（产品手册、客服FAQ、合同文本），用 RAG 就能让 AI 读自家文档来回答，准确率大幅提升，Token 消耗也会增加（读文档也要算 Token）。

什么是微调？客户需要微调吗？

微调 = 用自家业务数据继续训练 AI，让它更懂你的行业。就像新员工培训，让 AI 学你公司的话语风格和专业知识。

但微调成本高、周期长，一般只有用量极大（月亿级 Token）且有专属需求的客户才值得。大多数中小企业的需求，用好 Prompt（提示词）+RAG 就够了，不用微调。别主动推微调，免得吓到客户。

销售关键认知： 你不需要记住所有模型的技术参数，只需要知道「哪类客户适合用哪类模型」，这就是你的专业价值所在。 绿色角标 = rhcloud 平台已接入，可直接向客户推荐。
